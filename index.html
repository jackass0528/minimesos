Mini Mesos

Testing infrastructure for Mesos frameworks

Simple

Start a Mesos cluster on the command line or via Java API.

How Mini Mesos works →

Isolated

Mesos master, slave and Zookeeper processes run in separate Docker containers.

Mini Mesos Container API →

Integrated

Mini Mesos exposes framework, state and task information to its Cluster State API.

Mini Mesos Cluster State API →

Get up and running in seconds.
